ü§ñ Voia AI Assistant

Voia is your AI-powered command-line companion. It can help you with:

  • Answering questions
  • Explaining commands
  • Providing code examples
  • Offering suggestions
# Set your API key
voia-setkey your-api-key-here

# Choose AI model
voia-model gpt-4

# Ask anything
voia explain how to use git rebase

# Clear conversation
voia-clear
